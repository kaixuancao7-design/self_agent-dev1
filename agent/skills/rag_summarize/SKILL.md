# RAG Summarize Skill

## Overview
检索知识库中的相关文档并返回格式化的文档内容，用于回答用户问题。

## Capability
- **核心能力**: 基于向量数据库的语义检索
- **输入**: 用户查询字符串
- **输出**: 格式化的参考文档内容

## Parameters

| 参数名 | 类型 | 必需 | 说明 |
|--------|------|------|------|
| query | string | 是 | 用户的查询问题 |

## Returns
```
参考资料1：文档内容...
参考资料2：文档内容...
...
```

## Usage Example
```json
{
  "name": "rag_summarize",
  "parameters": {
    "query": "AI Agent 最近的热点趋势"
  }
}
```

## Use Cases
- 回答与知识库相关的问题
- 检索特定主题的参考资料
- 为用户查询提供背景信息

## Notes
- 如未检索到相关资料，返回"未检索到相关资料"
- 检索结果按相关性排序，最多返回3条
